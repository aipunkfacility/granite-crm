## Справочник плейсхолдеров

| Плейсхолдер | Описание | Пример подстановки |
|---|---|---|
| `{from_name}` | Имя отправителя | Александр |
| `{city}` | Название города (именительный падеж) | Екатеринбург |
| `{city_locative}` | Название города в предложном падеже (после предлога «в») | Екатеринбурге |
| `{original_subject}` | Тема исходного письма (для Re:) | Вопрос по ретуши |
| `{unsubscribe_url}` | Ссылка на отписку | https://… |

**Важно:** После предлога «в» всегда используйте `{city_locative}`, а не `{city}`. Склонение обеспечивает модуль `granite/city_declensions.py` с данными из `data/city_declensions.json` (1093 города). Если город отсутствует в словаре, используется именительный падеж как fallback.

***

**Технический совет:** После обновления JSON-файла не забудьте выполнить команду для «горячей» перезагрузки шаблонов в CRM:
`uv run cli.py templates-reload` (или вызов API). 

Теперь вся ваша коммуникация выглядит как единая профессиональная система. Удачи с первым запуском!
