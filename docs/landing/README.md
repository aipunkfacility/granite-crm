## Связь с CRM

| Поток | Как работает |
|-------|-------------|
| CRM → Лендинг | Email-шаблоны ссылаются на `retouchgrav.netlify.app` как на сайт компании |
| Лендинг → Контакты | TG/WA кнопки ведут на прямые чаты, не в CRM. CRM нужна для исходящего аутрича |
| Email-шаблон | `monument-web/email/email-improved-light.html` — исходник, загружается в CRM через файловый аплоад |
| Изображения | `memorial-img.netlify.app` — отдельный Netlify-сайт с картинками для email-шаблонов |
| Аналитика | Яндекс.Метрика (счётчик 108345194, 14+ целей). Не интегрирована с CRM |

### Что менять на лендинге при изменениях в CRM

| Изменение в CRM | Что обновить на лендинге |
|----------------|------------------------|
| Изменились цены | Секция Pricing |
| Изменились условия работы | Секция FAQ |
| Обновился HTML-шаблон письма | `monument-web/email/email-improved-light.html` |
| Добавились кейсы/отзывы | Секция Portfolio или новая секция |

## Дизайн-система (RetouchGrav DS v3.1)

### Ключевые цвета (Mineral Palette)

| Имя | Цвет | Hex | Где используется |
|-----|------|-----|-----------------|
| Labradorite | Фиолетовый | #7C8CF8 | Акценты, ссылки, активные элементы |
| Malachite | Зелёный | #34D399 | Успех, позитивные индикаторы |
| Amber | Золотой | #FBBF24 | Предупреждения, выделение цен |
| Garnet | Красный | #F43F5E | Ошибки, удаление, спам |

### Шрифты

- **Outfit** — заголовки (display font)
- **Inter** — основной текст (body font)

Оба шрифта используются и на лендинге, и в CRM — единое визуальное ощущение.

### Тема

Dark / Light / System — переключатель в навигации. Выбор сохраняется в `localStorage` под ключом `retouchgrav-theme`.

### Glassmorphism

На лендинге используется эффект «стекло»: полупрозрачные карточки с `backdrop-filter: blur()` и тонкой границей. В CRM — shadcn/ui компоненты без glassmorphism (другие паттерны для админ-панели), но с той же цветовой палитрой.

**Полная дизайн-система:** `monument-web/DESIGN_SYSTEM.md` (в репозитории лендинга)
**CRM дизайн-система:** [frontend/design-system.md](../frontend/design-system.md)
