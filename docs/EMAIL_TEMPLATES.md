# Email-шаблоны: Granite CRM

> Каталог всех email-шаблонов для рассылок и post-reply ответов.
> Тексты правит Александр. Загрузка в CRM: `uv run cli.py db seed-templates`
>
> Каждый шаблон хранится в БД как `CrmTemplateRow` с уникальным `name`.
> Плейсхолдеры: `{city}`, `{company_name}`, `{from_name}`, `{unsubscribe_url}`

## 4. Волновые темы для A/B тестирования

При создании кампании можно задать `subject_a` и `subject_b` для A/B теста. Тело письма одно — `template_name`.

### Волна 1: Основной оффер

| Вариант | Тема | Гипотеза |
|---------|------|----------|
| A | «Помощь с подготовкой фото для гравировки, {city}» | Нейтральная, профессиональная |
| B | «Сложные фотографии для гравировки — {city}» | Акцент на боль (сложные фото) |

### Волна 2: Сложные случаи

| Вариант | Тема | Гипотеза |
|---------|------|----------|
| A | «Ретушь портретов для мастерских — {city}» | Конкретная услуга |
| B | «Бесплатный тест: покажу результат за 2 часа» | Акцент на бесплатный тест |

### Волна 3: Социальное доказательство

| Вариант | Тема | Гипотеза |
|---------|------|----------|
| A | «Ретушь портретов для гравировки — уже работаем с мастерскими в {city}» | Социальное доказательство |
| B | «Мастерские в {city} экономят время на ретуши» | Выгода |

## 5. Порядок загрузки в CRM

```bash
# 1. Проверить текущие шаблоны в БД
uv run cli.py db list-templates

# 2. Загрузить все шаблоны из этого документа
uv run cli.py db seed-templates

# 3. Проверить что загрузилось
uv run cli.py db list-templates

# 4. Обновить один шаблон (после правки текста)
uv run cli.py db seed-templates --only reply_interested
```
